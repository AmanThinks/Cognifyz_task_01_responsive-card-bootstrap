# Responsive Card Layout using Bootstrap

This project was developed as part of my **Web Development Internship at Cognifyz Technologies**.

The objective of this task was to use a front-end framework to create a **responsive card component** that includes images and text while implementing a responsive grid layout.

---

## 📌 Task Objective

* Use a front-end framework like **Bootstrap**
* Create responsive **card components**
* Display **image and text content**
* Implement a **responsive grid layout**

---

## 🚀 Features

* Responsive card layout using Bootstrap
* Six cards with images and descriptions
* Smooth hover animation effects
* Animated title underline effect
* Dark themed UI design
* Fully responsive layout for different screen sizes

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Bootstrap 5
